### Kampüs Etkinlik Uygulaması



Proje Adı



Kampüs Etkinlik Uygulaması



Proje Amacı



Bu projenin amacı, kampüs içerisinde düzenlenen etkinliklerin merkezi bir sistem üzerinden yönetilmesini sağlamak ve kullanıcıların bu etkinliklere erişimini kolaylaştırmaktır. Sistem kapsamında kullanıcıların etkinlikleri görüntüleyebilmesi, etkinliklere katılabilmesi ve katılımlarını iptal edebilmesi; yönetici rolünün ise etkinlik oluşturma, düzenleme ve silme işlemlerini gerçekleştirebilmesi amaçlanmaktadır.



Kullanılan Teknolojiler



\- Python

\- Flask

\- HTML

\- CSS

\- Bootstrap

\- XAMPP üzerinde çalışan MySQL veritabanı

\- Git ve GitHub



Proje Demosunun Kurulum ve Çalıştırma Talimatı



1\. XAMPP uygulaması üzerinden Apache ve MySQL servislerini başlatınız.

2\. phpMyAdmin üzerinden proje için kullanılacak veritabanını oluşturunuz.

3\. Proje klasörünü bilgisayarınıza indiriniz veya kopyalayınız.

4\. Gerekli Python paketlerini kurunuz.

5\. Flask uygulamasını çalıştırınız.

6\. Tarayıcı üzerinden uygulamaya erişiniz.



Proje Yapısı



\- "docs/" klasörü: Gereksinim analizi ve UML diyagramları

\- "src/" klasörü: Projeye ait kaynak kodlar

\- "README.md": Proje açıklaması ve çalıştırma bilgileri
